## About
![Student view.](/images/portfolio/sikomodasa-2.png)
This website is designed as a communication tool between teachers, students, and parents to coordinate student behavior at school.

The system will maintain records of school regulations (managed by the admin), which students can easily check online. Furthermore, if a student commits an act that results in a change in their points, a teacher will record this through the application and explain the impact of the change on existing regulations.
![Student view.](/images/portfolio/sikomodasa-3.png)

Because all changes in student points are recorded in the application, students/guardians can review student behavior.

### Features
- **Role-Based Access Control:** Each user has limited influence within the application.
- **Responsive Design:** Web interface optimized for both desktop and mobile viewing.
